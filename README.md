## Overview

A collaborative front-end app that helps users search shows, browse trending titles, read summaries, and maintain a personal viewing schedule.

## Features

- 🔎 TV show search
- 📈 Trending show discovery
- 🧾 Show summaries and metadata
- 🗓️ Personal schedule list
- 💾 Local browser storage
- 📱 Responsive Bulma interface

## Tech Stack

- HTML
- CSS
- JavaScript
- Bulma
- Trakt API
- TVMaze API
